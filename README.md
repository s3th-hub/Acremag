# ACREMAG - Real Estate Management Platform

ACREMAG is a web-based real estate management platform designed to help property managers, landlords, and real estate businesses manage properties, tenants, and financial operations in a centralized system.

## 🔗 Overview
The platform simplifies real estate operations by combining property listings, tenant management, lease tracking, and financial handling into one intuitive system. It is built to improve efficiency, reduce manual processes, and provide better visibility into property performance.

## 🚀 Features
- Property listing and management
- Tenant onboarding and records management
- Lease tracking (start, renewal, termination)
- Rent collection and payment tracking
- Financial management (deposits, charges, recurring payments)
- Dashboard with key property insights
- Organized reporting for property and financial data

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
